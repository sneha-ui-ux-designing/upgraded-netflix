Netflix Mood Recommendation System Concept

This is a beginner UX/UI concept project exploring a mood-based recommendation system for streaming platforms

##Problem
Users often struggle to decide what to watch and keep switching between streaming apps due to overwhelming recommendations and decision fatigue.

##Solution
This concept introduces a guided recommendation flow where users answer simple questions based on-
-mood
-available time
-language
-energy level
-viewing preference

The recommendations adapt according to the user's selections to make content discovery easier and more personalized.The top picks changes each time as the options gets more refined and narrows problems.
